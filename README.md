# Serious Possibilists

##### (Spiced Academy Graduation Project)

### The Project

Serious Possibilists is an SPA that lets members register good deeds projects and track their contributions, individually and as a team. The idea is that every little bit helps, and if you track every little bit, it adds up!

URL: (currently down for improvements, can be activated on request)
https://serious-possibilists.herokuapp.com/

### Technology

-   React
-   Node.js, Express.js
-   Socket.io
-   PostgreSQL
-   AWS: SES, S3

### Main Features

-   Registration, login, reset password (limited functionality until I get wider SES privileges)
-   Profile blurb, picture, with projects and contributions
-   Project Registration, with name, category, metric, and recommended resources
-   People search and friend function
-   Project summaries are also visible to non-members
